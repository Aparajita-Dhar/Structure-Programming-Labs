# Structured Programming Paradigms in C

This repository showcases the implementation of structured programming concepts. It moves past simple scripting to explore programmatic flow efficiency, functional modularization, precision pointer arithmetic, and custom structural data types.

##  Repository Index

### Phase 1: Core Mechanics & Controls
* **`01_operators_and_arithmetic.c`**
  * Demonstrates arithmetic expressions, precedence evaluations, and comparative relational/logical operations.
* **`02_selection_and_assignments.c`**
  * Evaluates variable conditional selections, mutation assignments (`+=`), and memory side-effects of pre vs. post-increments.
* **`03_advanced_loops_and_switches.c`**
  * Contains complex multi-selection conditional routing using `switch-case` alongside precision iteration control (`do-while`, `break`, `continue`).
    ### Phase 2: Modular Systems & Data Processing
* **`04_program_modules_and_recursion.c`**
  * Implements modular programming through custom functions, handling scope parameters, storage traits (`static`), and execution stacks via mathematical recursion.
* **`05_multidimensional_arrays.c`**
  * Breaks down the structural management and iterative processing loops required for both 1D matrices and multi-dimensional 2D matrix grids.
* **`06_array_processing_and_algorithms.c`**
  * Demonstrates standard data manipulation techniques including passing reference arrays into modular functions, running element arrays through sorting passes (Bubble Sort), and running key-matching routines (Linear Search).
    ### Phase 3: Memory Routing & Complex Architectures
* **`07_pointer_fundamentals.c`**
  * Explores foundational pointer design patterns, memory address assignment mappings (`&`), indirection lookups, and direct value mutations via dereferencing tools (`*`).
* **`08_basic_structures.c`**
  * Builds multi-type composite data structures (`struct`), defining custom abstract models to package and process related record attributes.
* **`09_advanced_structures_and_unions.c`**
  * Features advanced records management layouts including nested schemas, structure reference pointers via arrow routing operators (`->`), and handles overlapping memory allocation envelopes via data `unions`.
